# 🚀 Quick Start - Deploy in 3 Minutes

## Option 1: Vercel Drag & Drop (Fastest)

1. **Download** this entire folder to your computer
2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
3. **Drag and drop** this folder onto the Vercel dashboard
4. Click **Deploy**
5. Done! Your dashboard is live 🎉

---

## Option 2: GitHub + Vercel (Recommended)

### Step 1: Create GitHub Repo
1. Go to [github.com/new](https://github.com/new)
2. Name: `geopolitical-dashboard`
3. Click **Create repository**

### Step 2: Upload Files
On your new repo page, click **"uploading an existing file"**
Drag and drop all files from this folder
Click **Commit changes**

### Step 3: Deploy to Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Select your `geopolitical-dashboard` repo
3. Click **Deploy**

---

## Option 3: Command Line

```bash
# Navigate to this folder
cd /path/to/this/folder

# Create GitHub repo and push
git init
git add -A
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/geopolitical-dashboard.git
git push -u origin main

# Then go to vercel.com and import your repo
```
